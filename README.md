# MentoRef Web App

MentoRef is a modern web application built with Go, HTMX, MongoDB, and Tailwind CSS.

## Features (WIP)

- User authentication - Sign up, sign in, and sign out functionality with JWT-based authentication
- Dashboard - A dashboard for users
- Real-time interactivity - The app uses HTMX to provide real-time interactivity without the need for JavaScript

## Technologies

- The backend is built with Go
- The app uses MongoDB as its database
- The frontend uses HTMX
- The app is styled with Tailwind CSS
